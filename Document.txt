1. PROJECT OVERVIEW



1.1 Purpose



A web-based system to digitally manage welder qualifications, certifications, and continuity records in compliance with welding standards (ASME, AWS, API). The system replaces manual paperwork with a searchable, traceable digital solution.



1.2 Target Users







Industrial Support Services Company administrators



QC managers and supervisors



Client representatives



Auditors







1.3 Key Objectives







Maintain single source of truth for 250-300+ welders



Reduce manual paperwork and errors



Enable fast audits & verifications



Generate professional certificates and reports



Track welder qualification continuity

14. SECURITY & AUTHENTICATION



14.1 Row Level Security (RLS)



sql-- All tables: Only authenticated users can access



CREATE POLICY "Authenticated users only"



  ON welders FOR ALL



  TO authenticated



  USING (true)



  WITH CHECK (true);



14.2 Storage Security



sql-- Allow authenticated uploads



CREATE POLICY "Authenticated uploads"



  ON storage.objects FOR INSERT



  TO authenticated



  WITH CHECK (bucket_id IN ('welder-photos', 'signatures', 'documents'));







-- Allow public reads



CREATE POLICY "Public reads"



  ON storage.objects FOR SELECT



  TO public



  USING (bucket_id IN ('welder-photos', 'signatures', 'documents'));
